# 🪞 Aptly Mirrors - Betriebsanleitung

**Playbook:** `pb_conf_aptly_mirrors_repo_server.yml`

---

## 🛠️ BEFEHLE

**Mirrors auflisten:**
```bash
aptly mirror list
```

**Mirror aktualisieren:**
```bash
aptly mirror update <mirror-name>
```

**Mirror erstellen:**
```bash
aptly mirror create <name> <url> <distribution>
```
